
# HNG Stage 0 API

Public API for HNG12 Internship Stage 0 Task

## For installation
`npm install`

## To start the server
`npm run start:dev`

## API Documentation

**Endpoint**: `GET /api`

**Response Format**:
```json
{
  "email": "string",
  "current_datetime": "ISO8601",
  "github_url": "string"
}
